# Plan: Tmux Session/Window Search Improvements

## Context
The setup already has `_tss_function` (session picker, `prefix+S` / `tss`) and `tmux-sessionx` (`prefix+o`).
The gaps are:
1. `_tss_function` is blind — no preview of what windows are in a session, and no way to kill a session from the picker.

### 2. New `_twf_function` — cross-session window fuzzy picker

Lists every window across every session as `session:index  window_name`. Selecting one switches to that session then that window.

**File to create:** `home-manager/programs/fish/functions/_twf_function.fish`

```fish
function _twf_function --description "Fuzzy-pick any window across all sessions"
  set -l selected (tmux list-windows -a -F '#{session_name}:#{window_index}  #{window_name}' 2>/dev/null \
    | fzf --prompt="window> " --height=40% \
          --preview='set t (string split "  " {})[1]; tmux list-panes -t $t -F "#P: #{pane_current_command}  #{pane_current_path}" 2>/dev/null')

  if test -z "$selected"
    return
  end

  set -l target (string split '  ' $selected)[1]   # "session:index"
  set -l parts (string split ':' $target)
  set -l sess $parts[1]
  set -l widx $parts[2]

  if test -n "$TMUX"
    tmux switch-client -t "$sess"
    tmux select-window -t "$sess:$widx"
  else
    tmux attach-session -t "$sess" \; select-window -t "$sess:$widx"
  end
end
```

Register in **`home-manager/programs/fish/default.nix`**:
- Add `"_twf_function"` to the `xdg.configFile` function list (alphabetical between `_tss_function` and `_two_function`)
- Add `twf = "_twf_function";` to `shellAbbrs`

### 3. `tmux.conf` — last-session toggle + window picker bind

Add two lines to **`home-manager/programs/tmux/tmux.conf`** in the bindings section:

```
# Last session (quick flip)
bind Tab switch-client -l

# Cross-session window picker
bind W run-shell "tmux new-window 'fish -c _twf_function'"
```

`Tab` is currently unbound. `W` is currently unbound (lowercase `w` is default select-window by index, not bound here).
