# Plan: Tmux Session History + `two` Resurrect Fix

## Context

Three related improvements to the tmux workflow:

1. **`_two_function` resurrect gap** — `_two_function` only runs resurrect restore when the work session does _not_ exist. `_tss_function` (work branch) runs restore _first_, then checks existence. `_two_function` should match that behavior so reconnecting always replays any saved state.

2. **Session history log** — No persistent record of which sessions/windows were active. Adding a 30-second background snapshot to a log file enables "where was I?" lookups via fzf.

3. **`_tsh_function` search** — A fish function to fzf-search the history log and optionally jump to a still-existing session/window.

## Changes

### 1. Fix `_two_function` — always resurrect first

**File:** `home-manager/programs/fish/functions/_two_function.fish`

Move the resurrect restore call to _before_ the session-existence check:

```fish
function _two_function --description "Attach to tmux work session"
  set -l restore (tmux list-keys 2>/dev/null | string match -rg '(/\S+/resurrect/scripts/restore\.sh)')
  set -l restore $restore[1]
  if test -n "$restore"
    tmux run-shell "$restore"
  end

  if tmux has-session -t work 2>/dev/null
    if test -n "$TMUX"
      tmux switch-client -t work
    else
      tmux attach-session -t work
    end
  else
    tmuxinator start work
  end
end
```

### 2. Session logger script

**File to add via `home.file`:** `~/.config/tmux/session-logger.sh`

Declare in **`home-manager/programs/tmux/default.nix`** using `home.file`:

```nix
home.file.".config/tmux/session-logger.sh" = {
  executable = true;
  text = ''
    #!/usr/bin/env sh
    mkdir -p ~/.local/share/tmux
    LOG=~/.local/share/tmux/session-history.log
    while true; do
      sleep 30
      tmux list-windows -a \
        -F "$(date +%Y-%m-%dT%H:%M:%S)  #{session_name}:#{window_index}  #{window_name}  #{pane_current_path}" \
        >> "$LOG" 2>/dev/null
      # Trim to last 10 000 lines to cap file size
      if [ "$(wc -l < "$LOG")" -gt 10000 ]; then
        tail -n 10000 "$LOG" > "$LOG.tmp" && mv "$LOG.tmp" "$LOG"
      fi
    done
  '';
};
```

Start it when the tmux server boots — add to **`home-manager/programs/tmux/tmux.conf`**:

```
# Persistent session history logger (starts once per server boot)
set-hook -g 'server-startup' 'run-shell -b "~/.config/tmux/session-logger.sh"'
```

### 3. New `_tsh_function` — search session history

**File to create:** `home-manager/programs/fish/functions/_tsh_function.fish`

```fish
function _tsh_function --description "Search tmux session history log"
  set -l log ~/.local/share/tmux/session-history.log
  if not test -f "$log"
    echo "No session history found at $log"
    return
  end

  set -l selected (cat "$log" | fzf \
    --prompt="session-history> " \
    --height=40% \
    --tac \
    --no-sort \
    --preview='echo {}')

  if test -z "$selected"
    return
  end

  # Field 2 is "session:window_index"
  set -l target (string split '  ' $selected)[2]
  set -l parts (string split ':' $target)
  set -l sess $parts[1]
  set -l widx $parts[2]

  if not tmux has-session -t "$sess" 2>/dev/null
    echo "Session '$sess' no longer exists"
    return
  end

  if test -n "$TMUX"
    tmux switch-client -t "$sess"
    tmux select-window -t "$sess:$widx" 2>/dev/null
  else
    tmux attach-session -t "$sess" \; select-window -t "$sess:$widx"
  end
end
```

Register in **`home-manager/programs/fish/default.nix`**:

- Add `tsh = "_tsh_function";` to `shellAbbrs`
