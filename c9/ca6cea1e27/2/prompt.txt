# Plan: Tmux Sessionizer with Default Sessions

## Context
The dotfiles already have 4 named tmux sessions managed individually via fish functions:
- `tpo` → `primary`, `tmo` → `mobile`, `tdo` → `desktop`, `two` → `work`

The goal is a **sessionizer** — an fzf-powered picker that lists default sessions, creates them if missing, and switches to them. This replaces manually typing `tpo`/`tmo` etc. and ensures default sessions always exist.

Existing relevant pieces:
- `home-manager/programs/fish/functions/_t{po,mo,do,wo}_function.fish` — individual session attach fns
- `home-manager/programs/fish/default.nix` — shellAbbrs + xdg.configFile function list
- `home-manager/programs/tmux/tmux.conf` — tmux keybinds, `tmux-sessionx` bound to `o`
- `tmux-resurrect` + `tmux-continuum` already configured for session persistence

## Implementation

### 1. New fish function: `_tmux_sessionizer_function.fish`

`home-manager/programs/fish/functions/_tmux_sessionizer_function.fish`

```fish
function _tmux_sessionizer_function --description "Fuzzy-pick or create a tmux session"
  set default_sessions primary mobile desktop work

  # Build candidate list: default sessions first, then existing sessions not in default list
  set existing (tmux list-sessions -F '#S' 2>/dev/null)
  set candidates $default_sessions
  for s in $existing
    if not contains $s $default_sessions
      set -a candidates $s
    end
  end

  set selected (printf '%s\n' $candidates | fzf --prompt="session> " --height=40%)
  test -z "$selected" && return

  if tmux has-session -t "$selected" 2>/dev/null
    if test -n "$TMUX"
      tmux switch-client -t "$selected"
    else
      tmux attach-session -t "$selected"
    end
  else
    tmux new-session -d -s "$selected"
    if test -n "$TMUX"
      tmux switch-client -t "$selected"
    else
      tmux attach-session -t "$selected"
    end
  end
end
```

### 2. Register abbreviation in `default.nix`

Add to `shellAbbrs`:
```nix
ts = "_tmux_sessionizer_function";
```

Add to the `xdg.configFile` function list:
```nix

```

### 3. Bind to tmux key in `tmux.conf`

Add a keybind so it's accessible from inside tmux (replaces or complements `tmux-sessionx`):
```
bind S run-shell "tmux new-window 'fish -c _tmux_sessionizer_function'"
```

This opens the sessionizer in a temporary window, switches session, then closes.
