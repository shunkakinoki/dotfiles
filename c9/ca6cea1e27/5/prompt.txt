# Plan: Tmux Session Presets via tmuxinator

## Context
The 4 named sessions (`primary`, `mobile`, `desktop`, `work`) are created bare with `tmux new-session -A -s <name>`. The goal is to have each session automatically start with predefined windows when first created (editor, shell, server, etc.). **tmuxinator** is the chosen mechanism — it reads a YAML layout file per session and wires up windows/panes on first create, then falls back to normal attach on subsequent invocations.

Pattern from the repo:
- `config/` = XDG config file management (e.g. `config/jj/`, `config/ghostty/`)
- `home-manager/programs/` = package + program option configuration (e.g. `home-manager/programs/tms/`)

### 4. Update the 4 individual fish functions

Pattern: if session exists → attach/switch normally; if new → `tmuxinator start <name>` (tmuxinator handles attach/switch-client automatically by detecting `$TMUX`).

**`_tpo_function.fish`**:
```fish
function _tpo_function --description "Attach to tmux primary session"
  if tmux has-session -t primary 2>/dev/null
    if test -n "$TMUX"
      tmux switch-client -t primary
    else
      tmux attach-session -t primary
    end
  else
    tmuxinator start primary
  end
end
```

Same pattern for `_tmo_function` (`mobile`), `_tdo_function` (`desktop`), `_two_function` (`work`).

### 5. Update `_tss_function.fish` (sessionizer)

Default sessions use tmuxinator on first create; ad-hoc sessions use plain `tmux new-session`:

```fish
function _tss_function --description "Fuzzy-pick or create a tmux session"
  set -l default_sessions primary mobile desktop work

  set -l existing (tmux list-sessions -F '#S' 2>/dev/null)
  set -l candidates $default_sessions
  for s in $existing
    if not contains $s $default_sessions
      set -a candidates $s
    end
  end

  set -l selected (printf '%s\n' $candidates | fzf --prompt="session> " --height=40%)

  if test -n "$selected"
    if tmux has-session -t "$selected" 2>/dev/null
      if test -n "$TMUX"
        tmux switch-client -t "$selected"
      else
        tmux attach-session -t "$selected"
      end
    else if contains $selected $default_sessions
      tmuxinator start "$selected"
    else
      tmux new-session -d -s "$selected"
      if test -n "$TMUX"
        tmux switch-client -t "$selected"
      else
        tmux attach-session -t "$selected"
      end
    end
  end
end
```
